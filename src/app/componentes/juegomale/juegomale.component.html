Juego para Malena

Juego de escribir palabras. Pensado para ayudar a mi hija, mientras aplico lo aprendido en este curso. Consiste en escribir una a una las letras de cada palabra segun la imagen. Sin posibilidad de equivocarse de letra. Entre los componentes del porfolio agregé el siguiente: "JuegoMaleComponent". Se usó TypeScript, Javascript y e Inyeccion de un servicio (que es el que genera el random) en dicho componente.
